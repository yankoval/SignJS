# SignJS — Инструмент для создания ЭЦП (CAdES-BES)

Простое и надежное веб-приложение для формирования электронной подписи локальных файлов через браузер. Проект разработан для упрощения работы с системой **«Честный ЗНАК»** (ГИС МТ) и другими сервисами, требующими формат подписи PKCS#7 (CAdES-BES).

## Основные возможности
- **Два режима подписи:** - **Отсоединенная (Detached):** Создается файл `.sig`, содержащий только криптографический хвост (идеально для ГИС МТ).
  - **Встроенная (Attached):** Исходный файл упаковывается внутрь подписи.
- **Умный поиск сертификатов:** Скрипт проверяет как личное хранилище пользователя (`Current User`), так и хранилище компьютера (`Local Machine`).
- **Безопасность:** Файлы не загружаются на сервер. Вся криптография выполняется локально через расширение КриптоПро.

## Требования к системе
Для корректной работы на компьютере должны быть установлены:
1.  **КриптоПро CSP** (версия 4.0 и выше).
2.  **КриптоПро ЭЦП Browser Plug-in** (версия 2.0).
3.  **Расширение для браузера** (Яндекс.Браузер, Chrome, Edge или Opera).

## Инструкция по запуску
1. Скачайте файлы проекта: `signJs.html`, `SignJsApp.js`, `style.css`.
2. Поместите библиотеку `cadesplugin_api.js` (от КриптоПро) в ту же папку.
3. Откройте `signJs.html` в браузере.
4. **Важно:** Если вы открываете HTML-файл напрямую с диска (в адресной строке `file:///...`), обязательно зайдите в настройки расширения КриптоПро и включите опцию **«Разрешить открывать ссылки на файлы»**.

## Структура файлов
- `signJs.html` — Структура интерфейса.
- `SignJsApp.js` — Логика взаимодействия с плагином и обработки файлов.
- `style.css` — Стили оформления.

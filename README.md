# SignJS — Инструмент для создания отсоединенной ЭЦП (CAdES-BES)

Простое веб-приложение для подписания локальных файлов с использованием криптопровайдера **КриптоПро CSP**. Идеально подходит для подготовки файлов к отправке в систему **«Честный ЗНАК»** (ГИС МТ).

## Особенности
- **Отсоединенная подпись (Detached):** Создает файл подписи отдельно от исходного файла.
- **Безопасность:** Подписание происходит локально на вашем компьютере через браузерный плагин.
- **Поддержка Local Machine:** Автоматически ищет сертификаты как в личном хранилище пользователя, так и в хранилище компьютера.
- **Сохранение в .sig:** Удобная выгрузка результата в готовый файл.

## Требования
Для работы приложения на компьютере должны быть установлены:
1. [КриптоПро CSP](https://www.cryptopro.ru/products/csp) (с действующей лицензией).
2. [КриптоПро ЭЦП Browser Plug-in](https://www.cryptopro.ru/products/cades/plugin).
3. Расширение для браузера (Chrome/Yandex/Edge).

## Как запустить
1. Склонируйте репозиторий или скачайте файлы.
2. Убедитесь, что файл `cadesplugin_api.js` находится в корневой папке.
3. Откройте `signJs.html` в браузере (рекомендуется **Яндекс.Браузер**).
4. **Важно:** Если вы открываете файл напрямую с диска (`file://`), в настройках расширения «КриптоПро» в браузере должна быть включена опция **«Разрешить открывать ссылки на файлы»**.

## Структура проекта
- `signJs.html` — интерфейс приложения.
- `SignJsApp.js` — логика взаимодействия с плагином.
- `style.css` — стили оформления.
- `cadesplugin_api.js` — официальная библиотека от КриптоПро.
